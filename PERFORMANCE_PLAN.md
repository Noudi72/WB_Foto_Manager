# Performance & Design Upgrade Plan

## Ziel: Professionelle Sportfotografie-App wie Lightroom Classic

### 1. Performance-Optimierung (KRITISCH)
- [ ] Intelligentes Prefetching (lädt nächste 5 Bilder im Hintergrund)
- [ ] Progressive Loading (Low-Res → High-Res)
- [ ] Smart Cache (behält aktuelle + nächste 10 Bilder im RAM)
- [ ] Background Loading (alle I/O auf Background Threads)
- [ ] OneDrive-Optimierung (lokales Caching, Prefetch)

### 2. Hochauflösende Vorschau
- [ ] 4K Vorschau (3840x2160) für Agentur-Check
- [ ] 100% Ansicht für Schärfe-Prüfung
- [ ] Schnelles Zoom (ohne Nachladen)
- [ ] Smart-Loading (nur aktuelle Bilder in 4K)

### 3. Design-Upgrade (Lightroom Classic Style)
- [ ] Dunkles, professionelles Theme
- [ ] Optimierte Typografie (bessere Lesbarkeit)
- [ ] Toolbar wie Lightroom (kompakt, funktional)
- [ ] Sidebar wie Lightroom (minimalistisch)
- [ ] Filmstrip unten (wie Lightroom)

### 4. Workflow-Optimierung (Sportfotografie)
- [ ] Tastatur-Shortcuts (Rating 1-5, Pfeiltasten)
- [ ] Schnelle Navigation (keine Verzögerung)
- [ ] Auto-Advance (nach Rating zum nächsten Bild)
- [ ] Farb-Labels für Quick-Selection
- [ ] Batch-Rating (mehrere Bilder markieren)

### 5. Kritische Funktionen
- [ ] Rating-System (funktioniert zuverlässig)
- [ ] Export (schnell, zuverlässig)
- [ ] IPTC (schnell bearbeiten)
- [ ] Crop (schnell, präzise)
- [ ] Adjustments (Echtzeit-Vorschau)

## Priorität: 1 → 3 → 4 → 5 → 2
