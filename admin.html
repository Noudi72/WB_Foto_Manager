GitHub Auto-Push (optional, aber empfohlen):

Wenn konfiguriert, wird gallery.json automatisch zu GitHub gepusht - kein manueller Download/Push mehr nötig!

  1. GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. WICHTIG: "Generate new token" → Name: "WB Foto Manager", Scope: repo (vollständiger Zugriff) - NICHT "Fine-grained token" verwenden!
  3. Token kopieren und unten eingeben (wird lokal gespeichert)
Häufiger Fehler: Wenn Sie einen "Fine-grained Personal Access Token" erstellt haben, funktioniert dieser nicht! Sie müssen einen "Tokens (classic)" mit repo Scope verwenden.
